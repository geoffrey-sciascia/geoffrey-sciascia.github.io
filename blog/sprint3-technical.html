An analogy to describe the differences between HTML and CSS

I would liken HTML and CSS to phtosynthetic Coralites and Zooxanthillae respectively. Coral build their own structures and house the Zooxanthillae in their flesh which surround the structure. And in turn, the Zooxanthillae provide the coral with nutrients, and colour the coral. Neither can survive without the other, and if there is something wrong with either, both animals die. Basically the Zooxanthillae provide content to the Coral, and the Coral provides structure for the Zooxanthillae.
